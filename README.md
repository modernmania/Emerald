> Emerald is a high-level programming language written in Python, with Golang for the terminal created by **@AModernAnimator** & **@Sushi-byte-glitch**. It is designed for **backend**, **frontend**, and **data**.

### Installation
  Install the .exe or tar.gz Emerald Shell from the Releases.


### Emerald Shell Commands :
find example.emer     = Make new file
carve example.emer    = Edit file  
shine example.emer    = Run file

### Syntax
---
  Variables
  ```
var [
  exampleVar=5
]
```
Put this `var []` block at the beginning of your Emerald script.

  Output
  ```
    print"Hello!"  # Prints the given string.
    print{score}   # Prints the variable's value.
    print{score}+5 # Prints that variable's value plus five.
  ```
